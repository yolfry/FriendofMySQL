Friend of MySQL 1.0.1

Friend of MySQL is a simple php framework, which allows you to synchronize in an organized and formatted way the queries to the MySQL data base.

This Framework was developed for the applications and web platform in MVC, that is, model, view, controller.

This one focuses on the wall of the Model -> Backend. In this way it manages to organize the queries sent to the database, through an object-oriented programming (OOP).

How to implement? to your web

The first thing is that you have to have an MVC web structure in this way.

  • Model
    • FriendofMySQL /* This is the framework folder */
  • View
    • Mi.page.web.html /* This is the view of the web page*/
  • Controllers
    • element.php /* This is the controller of the website*/
  • index.php /* This is the index of the web page */

In this way, it will be the correct way to implement the framework on your web page.

How to configure? FriendofMySQL

To configure the name of the database:
In the File FriendofMySQL/config/name_db.php There is a variable:
$CONFIG['db']['name'][1] ="example";
"example" in this case it is the name of the database.


To configure the database server:
At FriendofMySQL/config/archivo server_db.php
These variables are to configure the server.
$CONFIG['db']['host'] = "localhost"; /*ip server*/
$CONFIG['db']['user'] = "root"; /*username of the server*/
$CONFIG['db']['pass'] = ""; /*server password*/

  • Model
    • FriendofMySQL
      • Config
        • name_db.php /*In this file you can configure the name of the database*/
        • server_db.php /*In this file you can configure the database server*/
  • View
  • Controllers
  • index.php

How to use? FriendofMySQL on your website

For the operation of FriendofMySQL after including the framework, some parameters are used. Example:

  • FriendofMySQL = new FriendofMySQL(-pass,-cmd,-data);
    -pass="root";
    -cmd="insert_example";
    -data['name']="example_name";

-pass : It is the parameter, so that the framework can confirm the password and start the procedure.
-cmd : Command or parameter id query FriendofMySQL/query/query.php.
-data : If a query block is made to insert a record or etc. The parameter data is sent, with the data that will be recorded eg $data['name']='yolfry';


Create a data record

To create a "data record" we have to register a new MySQL query block in the file FriendofMySQL/query/query.php ej. if($cmd ="insert_example"){
/*MySQL query*/
}
Within the query block, placed the following code:

    /*#File: FriendofMySQL/query/query.php*/
    if($cmd ="insert_example"){

    $sql = "INSERT INTO `example`(`id_example`, `name`) VALUES ('','".$data['name']."')";



    if(!$connection->query($sql)){

        $this->callback="false";

    }


    $this->callback="true";



    }

In the query block there are some variables and objects such as: $connection , $data , $this-> callback
$connection : This object contains the connection of the database, it is used to make queries.
$data : This works as a data transport variable that is invoked. it only works as Get in the query block.
$this-> callback :This is used to return the data of a select from the database, also to send true or false.

Once registered the query block, you just have to use it on your website.

  • We include the framework.

    <?php
       include("FriendofMySQL/FriendofMySQL.php");
    ?>

  • We begin with the registration.

    <?php

      $data['name']="example_name";

       $FriendofMySQL = new FriendofMySQL("root","insert_example",$data);
       if($FriendofMySQL->callback!="true"){
        echo "Complete example registration.";
       }else{
        echo "Error, could not register";
       }
    ?>

This way data is recorded with a query in Friend of MySQL


Create a data query

To create a "data query" we have to register a new MySQL query block in the file FriendofMySQL/query/query.php ej. if($cmd ="consul_example"){
/*Consulta Mysql*/
}
Within the query block, placed the following code:

    /*#File: FriendofMySQL/query/query.php*/
    if($cmd =="consul_example"){

      $sql ="SELECT * FROM `example`";
      $data = $connection->query($sql);
      $example = array();
      $array = array();
      $i=0;


     while($example = $data->fetch_array(MYSQLI_BOTH)){

      $array[$i]['id_example'] = strip_tags($example['id_example']);
      $array[$i]['name'] = strip_tags($example['name']);
      $i++;
     }
      $this->callback = $array; /* return parameter*/
    }

In the query block there are some variables and objects such as: $connection , $this-> callback
$connection : This object contains the connection of the database, it is used to make queries.
$this-> callback : This is used to return the data of a select from the database, also to send true or false.

Once registered the query block, you just have to use it on your website.

  • Incluimos el framework.

    <?php
       include("FriendofMySQL/FriendofMySQL.php");
    ?>

  • We begin to show the data on your website.

    <?php
       $FriendofMySQL = new FriendofMySQL("root","consul_example");
       if($FriendofMySQL->callback!="true"){
        for($i=0;$i < count($FriendofMySQL->callback);$i++){

          echo"<br>".$FriendofMySQL->callback[$i];

        }
       }else{
        echo "Error, could not show data";
       }
    ?>

In this way, data is consulted in "Friend of MySQL"
